**Telegram Bot для музея Experimentanium**  
Бот для предоставления информации о музее Experimentanium: контакты, расписание, экскурсии, дни рождения и полезные ссылки.

**Функционал бота:**  
Контакты - телефоны, email, Telegram  
Время работы - график работы музея  
Адрес и парковка - как добраться  
Экскурсии - виды экскурсий, стоимость 
Полезные ссылки - бронирование, афиша, лагерь  
Дни рождения - организация праздников в музее  
Цены и шоу - актуальная информация
